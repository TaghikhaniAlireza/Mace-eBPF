# Version notes — 0.1.x

This page records **user-visible** changes and integration notes for the **0.1.x** release line of Aegis-eBPF (agent, SDK, and packaging). Patch numbers follow repository tags when present (for example **`v0.1.0`**).

## 0.1.x highlights (repository state)

| Area | Capability |
|------|------------|
| **eBPF sensor** | Syscall tracepoints for memory-relevant events (`mmap`, `mprotect`, `execve`, `openat`, `ptrace`, `memfd_create`, …) with ring-buffer export and CO-RE-oriented layouts. |
| **Rust core** | Pipeline with reordering, enrichment, YAML rules + suppressions, `StandardizedEvent` JSON export, optional observability features behind Cargo features. |
| **FFI** | C ABI (`aegis.h`); **`staticlib`** + **`cdylib`** for Go static link and Python `ctypes`. |
| **Go SDK** | `clients/go/aegis`: **`AegisEvent`**, **`Client`**, **`NewClient`** / **`Close`**, channel-based delivery; log level wrappers. |
| **aegis-agent** | Cobra CLI, **`--config`**, file-backed logging (JSON/text), systemd-friendly shutdown. |
| **Packaging** | **nFPM** `.deb` (`packaging/nfpm.yaml`), lifecycle scripts, **`aegis.service`**. |
| **OCI** | Multi-arch **`ghcr.io/taghikhanialireza/aegis-ebpf`** (see [Quickstart](../../1-getting-started/quickstart.md)); cosign-signed digests on publish. |

## Upgrade notes

- **Rules YAML:** prefer explicit **`suppressions:`** for trusted noise rather than deleting detection rules; suppression preserves **`matched_rules`** in exported JSON while listing **`suppressed_by`**.
- **Go embedders:** migrate from a raw callback to **`aegis.NewClient`** + **`Events()`** for back-pressure-friendly delivery.
